Python Text Editor (Work in Progress)

A lightweight text editor built with Tkinter.

Current features:
- Open text files
- Simple editor interface
- Optional RTL typing support

Notes:
RTL typing is implemented using Unicode direction markers.
Tkinter's native Text widget has limited bidirectional editing support, which may cause cursor inconsistencies during inline editing.

Planned improvements:
- Saving text files
- Autosave
- Dark mode
- UI polish
- Code cleanup and refactoring
